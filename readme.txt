Init Bootstrap 4 to Wordpress.

== Description ==

Init Bootstrap 4 to Wordpress. So you can use all Bootstrap 4 classes in your theme and in your preferred editor.

== Frequently Asked Questions ==

= Do I need something else to use bootstrap? =

No you don't, the plugin is ready-to-go. Install it, activate it and you are ready for Bootstrap.

= Is it compatible with every theme? =

Theoretically yes, but it is demanded by your theme implementation.
